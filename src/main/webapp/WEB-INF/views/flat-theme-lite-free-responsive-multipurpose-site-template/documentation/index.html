Main File Structure

All the file are well organized, its so easy to work with the template.

1. Unzip the files.
2. Open "flatpro-file" folder.
3.You will find the all html file (Ex: index.html / **.html) on the root that will be edited.
Also you will see 5 directories, css, js, fonts, sass and img directories. In the css folder Content you will find main.css and other styling files the all the other files expect the htmls. Js folder contain only the jquery library. Fonts folder contain essetial font files for fonts. 

                main-path/
                └── css/
                    ├── ***.css
                    └── main.css
                └── sass/
                    ├── ***.sass
                    └── style.css
                ├── fonts
                ├── img
                └── js/
                        ├── ***.js
                        └── active.js
                |
                ├── ***.html
                └── index.html
